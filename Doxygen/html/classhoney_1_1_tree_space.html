Detailed Description

Space that is ordered by a hierarchy of bounding volumes.

Objects can be added through this interface or through the Tree component of any children or descendants.
Objects do not generally require a transform component. A tm is required only in certain situations like if culling is enabled.
The CullVol component is used to specify culling for an object.
