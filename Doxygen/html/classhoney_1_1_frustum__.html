Detailed Description

template<class Real>
class honey::Frustum_< Real >

3D frustum geometry base class

A frustum is defined by 6 bounding planes with inward facing normals.
These planes are created from 3 unit axes (forward, up, right) and min/max extents along those axes.

Forward axis = -Z axis
Up axis = Y axis
Right axis = X axis

minExtent.x and maxExtent.x define the near and far planes along the forward axis.
minExtent.y and maxExtent.y define the bottom and top planes along the up axis.
minExtent.z and maxExtent.z define the left and right planes along the right axis.
