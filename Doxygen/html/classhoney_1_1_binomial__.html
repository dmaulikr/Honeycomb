Detailed Description

template<class Real>
class honey::Binomial_< Real >

Generate a random integer variate from a binomial distribution.

The result x is the integer number of successes in n independent trials, where each trial has success probability p.

Example: Toss a coin (head p = 0.5) 50 times (n) and count number of heads (x)
