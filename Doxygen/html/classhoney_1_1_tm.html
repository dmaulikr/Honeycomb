Detailed Description

Transform component.

Not all objects in the scene graph require a transform component. Objects without a tm component are skipped in the world tm calculations.

const Transform& honey::Tm::world ( bool  ancestorsUpdated = false) const
inline

Get world transform. Also updates and caches this node's world tm, same as updateWorld().

There are two ways to call world():

Ancestors not updated (slower): Node will recursively check that parent world tms are up-to-date all the way to the root.
Ancestors updated (faster): Ancestors are assumed to be up-to-date, no recursive check is necessary.
This method can be used during a preorder traversal if world(true) is called on every node.
