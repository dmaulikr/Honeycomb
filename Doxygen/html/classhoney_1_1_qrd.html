Detailed Description

template<class Real>
class honey::Qrd< Real >

QR Decomposition. Can be used to solve least squares problems.

The reduced QRD of (m x n) matrix A with m >= n is: where Q is an orthogonal (m x n) matrix, and R is an upper triangular (n x n) matrix.

The full QRD is: where Q is (m x m) and 0 is (m-n x n).

Most linear systems do not require the full QRD to solve.
