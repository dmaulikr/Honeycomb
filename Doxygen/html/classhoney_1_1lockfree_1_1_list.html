Detailed Description

template<class Data_, class Alloc_ = std::allocator<Data_>, class Backoff = Backoff, int iterMax = 2>
class honey::lockfree::List< Data_, Alloc_, Backoff, iterMax >

Lock-free doubly-linked list.

Based on the paper: "Lock-free deques and doubly linked lists", Sundell, et al. - 2008

Template Parameters
DataContainer element type
AllocNode allocator. Pointers returned by allocator must be at least 2-byte aligned.
BackoffBackoff algorithm to reduce contention
iterMaxMax number of iterator instances per thread
