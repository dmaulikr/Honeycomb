Detailed Description

template<class Real>
class honey::Gamma_< Real >

Generate a random variate from a gamma distribution .

The gamma distribution is a core component for most other distributions. It is a distribution of the amount of time required for a number of events to occur – the opposite of a poisson distribution.

Example: Waiting time until death
